#  Vue Card-Pairs Game

## Infos

A Card-Pairs game implemented in Vue.js (Vue3, Script-Setup, Typescript).

Tech used: Vue.js + TypeScript + Vite + Tailwind

Images are from Pexels.com

## Install + Develop

To install/develop

```bash
npm i
npm run dev
```
